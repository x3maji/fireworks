Техническое задание для разработки сайта компании «Nova-salut»

Описание задачи

Разработать интернет-магазин для компании, занимающейся оптовой продажей фейерверков,
устраивающая пиротехнические шоу по заказу клиента при праздновании какого-либо события.
Наши потенциальные оптовые клиенты - владельцы розничных магазинов пиротехники и фейерверков,
с годовым объёмом закупок до 2 000 000 рублей.

Описание главной страницы

Страница состоит из шапки, и подвала.
Имеется логотип, главное меню, поиск по сайту.


В области контента могут находиться следующие элементы:
  • Главный баннер
  • Категории
  • Лучшие товары
  • Преимущества компании
  • Информация о магазине

В подвале: техника безопастности, сертификаты, отзовы,организация фейерверк шоу.


Главное меню должно включать:

  • О Компании
  • Каталог
  • Акции
  • Доставка и оплата
  • Организациям
  • Контакты

Требования к верстке

  • Кроссбраузерность
  • CSS,JS файлы всегда должны быть во внешних файлах
  • Все файлы кодируются в кодировке UTF-8 without BOM
  • Разделяйте CSS и основные html блоки комментариями
  • В папке с изображениями не должно быть картинок, не использующихся в верстке
  • Должен подключаться только один минифицированный файл css в начале HTML-кода и только один минифицированный файл JS в самом конце
  • Изображения, шрифты, java-скрипты, т.е. разные типы файлов, хранить в соответствующих папках

SEO-требования к сайту

  • У каждой страницы должны быть теги H1, title и description.
  • Все тексты, должны быть уникальные.
  • Все стили прописывать в отдельном файле CSS

Функционал и безопасность

  • Форма заказа в карточке товара
  • Поиск по сайту
  • Вывод товаров и его категорий из БД
  • Личный кабинет
